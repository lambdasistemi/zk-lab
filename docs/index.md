# zk-lab

Laboratory for zero-knowledge and privacy-preserving signature primitives on
Cardano.

Express statements once in a high-level intention-driven DSL; compile them to
Plutus-verifiable proofs across multiple backends — Groth16, BBS+, and Halo2 —
with feature parity as a design target.

Content is built out in subsequent PRs. This bootstrap PR wires CI so docs
can be previewed on surge.sh before merging.
